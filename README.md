2. **Rename** folder to `lxr-cleangun` (exact name required)
3. **Copy** to your `resources/` folder
4. **Import** the SQL file for your language:


## âš™ï¸ Configuration

LXR Weapon Cleaner is highly configurable. Key options include:

```lua
-- Framework (auto-detect or manual)
Config.Framework = 'auto'

-- Language
Config.Lang = 'en'

-- Cooldowns (seconds)
Config.Cooldowns = {
    cleanWeapon = 10,
    inspectWeapon = 2
}

-- Security
Config.Security = {
    enableCooldowns = true,
    maxCleaningsPerMinute = 10
}

-- Add cleaning items
Config.Items = {
    cleanshort = { dbname = "cleanshort", label = "Weapon Cloth" },
    cleankit = { dbname = "cleankit", label = "Cleaning Kit" }
}
```

See [Configuration Guide](docs/configuration.md) for all options.
